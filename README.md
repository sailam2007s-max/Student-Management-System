# 🎓 Student Management System

## 📌 About the Project
This is my first Python desktop application built using Python, Tkinter, and SQLite.

The project helps manage student records through a simple graphical user interface (GUI). It allows users to add, search, update, and delete student details efficiently.

## 🛠️ Technologies Used
- Python
- Tkinter
- SQLite

## ✨ Features
- ✅ Add Student
- ✅ Search Student
- ✅ Update Student
- ✅ Delete Student
- ✅ Clear Fields
- ✅ Total Students Count
- ✅ SQLite Database Integration
- ✅ User-friendly GUI
- ✅ Confirmation pop-up messages

## 🚀 How to Run
1. Download or clone this repository.
2. Open the project in Visual Studio Code.
3. Run `main.py`.
4. Start managing student records.
